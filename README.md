# ESG Emissions Dashboard

A full-stack ESG emissions tracking dashboard built using Django REST Framework and React.

## Features

* CSV Upload functionality
* ESG emissions analytics dashboard
* Total emissions calculation
* Suspicious records tracking
* REST API integration
* React frontend with Django backend

## Tech Stack

### Backend

* Python
* Django
* Django REST Framework
* Pandas
* SQLite

### Frontend

* React
* JavaScript
* Axios
* Vite

## Project Structure

backend/ → Django backend
frontend/ → React frontend
sample_data/ → CSV sample files
screenshots/ → Project screenshots

## How To Run

### Backend

cd backend
.\venv\Scripts\activate
python manage.py runserver

### Frontend

cd frontend
npm install
npm run dev
